ملاحظات مستقبلية للربط

هذه الصفحة ثابتة الآن ولا تستدعي أي API. عند الانتقال إلى الربط الفعلي يمكن تحويل البيانات التجريبية إلى مصدر مركزي يدعم التصفية والمراجعة والصلاحيات.

  • Google Sheets كمصدر أولي لقراءة الطلبات وتحديث حالة الشهادة.
  • n8n Webhook لإعادة الإرسال أو إعادة محاولة الإصدار من زر إداري واضح.
  • Firebase أو قاعدة بيانات لاحقًا لتخزين الشهادات، نسخ الاختبارات، وسجل الأخطاء.
  • تسجيل دخول إداري قبل الوصول للصفحة، مع صلاحيات للمشاهدة، إعادة الإرسال، وإدارة البرامج.
  • إضافة حقول للعميل أو الجهة عندما تصبح الشهادات مرتبطة بعقود أو برامج مؤسسية.
