Quels types de retours envoyer ?

Un message court suffit si l'idée est claire. Vous pouvez aussi préciser la page concernée.

Amélioration du site

Une idée pour rendre une page plus claire, simplifier un parcours ou améliorer l'expérience de navigation.

Nouvel outil

Une proposition d'outil pratique pour aider à structurer un objectif, un projet, une décision ou un suivi.

Modèle ou ressource

Un modèle, une checklist, un document ou une ressource que vous aimeriez voir ajouté à ConsuTrain.

Problème à signaler

Un lien qui ne fonctionne pas, un problème d'affichage, une erreur de contenu ou une difficulté d'utilisation.

Idée générale

Une suggestion plus large pour enrichir les services, les contenus, les formations ou les parcours utilisateurs.

Orientation utile

Si votre message concerne un besoin concret, vous pouvez aussi consulter les services ou envoyer une demande.
